## Program 11 : Write a code to get character variable of a ASCII value .
```C
#include<stdio.h>
int main()
{
    int i;
    printf("Enter a ASCII value: ");
    scanf("%d",&i);
    printf("Character value of %d =%c",i,i);
    return 0;
}
```C
** Output : Enter a ASCII value : 8
  Character value of 8 = D **
